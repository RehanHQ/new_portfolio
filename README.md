# Rehan Haque - AI Portfolio

![Portfolio Screenshot](https://static.photos/technology/1200x630/42)

Personal portfolio website showcasing AI projects, skills, and experience.

## Features

- Responsive design
- Interactive sections
- Dark mode (coming soon)
- Project showcase
- Contact form functionality

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Tailwind CSS
- Feather Icons

## Setup

1. Clone the repository:
```bash
git clone https://github.com/[your-username]/[repository-name].git
```

2. Open `index.html` in your browser.

## Deployment

This project is deployed using GitHub Pages. To deploy your own version:

1. Go to repository Settings
2. Select "Pages" from the sidebar
3. Choose the main branch and root folder
4. Click Save

Your site will be live at: `https://RehanHQ.github.io/new_portfolio`
